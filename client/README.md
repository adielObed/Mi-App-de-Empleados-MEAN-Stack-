Mi App de Empleados (MEAN Stack)

Esta es una aplicación sencilla que hice para practicar cómo conectar un servidor de Node.js con una base de datos de MongoDB y mostrar todo en una página con Angular. 

El objetivo es poder registrar personas y organizarlas por las áreas de una empresa.

¿Qué hace la aplicación?

-Gestionar Empleados: Puedes agregar, ver, editar y borrar personas de la lista.-Gestionar Departamentos: Puedes crear las áreas de la empresa (como Logística o Ventas).
-Ver el Equipo: Hay una sección donde puedes ver cada departamento y quiénes trabajan en él.

Tecnologías que uséAngular: Para toda la parte visual y los formularios.Node.js y Express: Para crear la API que maneja los datos.MongoDB: Para guardar la información de forma segura.

Datos que se guardan para cada empleado, la app pide estos datos obligatorios:

Código: Un número único para identificarlo.Nombre y Apellidos: Sus datos personales.Código de Departamento: Para saber en qué área trabaja.
